Welcome to joeQuery

To make a joeQuery object in the console, wrap an HTML element or CSS selector with $l( <- selector -> )

A lightweight library with the following features

  • DOM Manipulation
  • DOM Traversal
  • Event Handling
  • AJAX Requests

The below methods are at your disposal

  • each()
  • html()
  • empty()
  • append()
  • attr
  • addclass(className)
  • removeClass(className)
  • children()
  • parent()
  • find(selector)
  • remove()
  • on(event, callback)
  • off(event)
